## Quick Start for Admins

### 1. Access Admin Dashboard

1. Sign in at **admin.protocol-guide.com**
2. Use your admin credentials (provided during onboarding)
3. You'll see the department overview dashboard

### 2. Add Your First Users

1. Go to **Users** → **Add Users**
2. Choose method:
   - **Bulk invite** via CSV upload
   - **Email invite** for individuals
   - **SSO sync** (if configured)

### 3. Configure Department Settings

1. Go to **Settings** → **Department**
2. Set your region/county
3. Upload department logo (optional)
4. Configure default user preferences

### 4. Review Analytics

1. Go to **Analytics** → **Overview**
2. See usage metrics across your department
3. Export reports as needed
