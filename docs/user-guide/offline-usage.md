## How Offline Mode Works

Protocol Guide uses **service worker technology** to cache protocols on your device:

1. **Automatic Caching**: Critical protocols download in the background
2. **Smart Updates**: New versions sync when you're online
3. **Offline Detection**: App automatically switches to offline mode
4. **Full Functionality**: Search, view, and access bookmarked protocols

## Setting Up Offline Access

### Enable Offline Mode

1. Go to **Profile** ‚Üí **Settings** ‚Üí **Offline Mode**
2. Toggle **"Enable Offline Protocols"** to ON
3. Select which protocols to download:
   - **Essential Only** (~50MB) ‚Äì Critical emergency protocols
   - **My Region** (~150MB) ‚Äì All protocols for your selected county
   - **Full Database** (~500MB) ‚Äì All 55,000+ protocols (Pro only)
4. Tap **"Download Now"**
5. Keep the app open until download completes

### Storage Requirements

| Download Option | Size | Contents |
|-----------------|------|----------|
| Essential Only | ~50MB | Cardiac arrest, trauma, airway, common medications |
| My Region | ~150MB | All protocols for your county/state |
| Full Database | ~500MB | Complete national protocol database |

### Recommended: Download Over Wi-Fi

Large protocol downloads should be done on Wi-Fi to:
- Save cellular data
- Download faster
- Prevent interruptions

## Troubleshooting Offline Mode

### "Offline mode not working"

1. Ensure offline mode is **enabled** in settings
2. Check that protocols have **finished downloading**
3. Try closing and reopening the app
4. Check your device's storage space

### "Protocols not downloading"

1. Connect to a **strong Wi-Fi signal**
2. Ensure you have **sufficient storage** (check Device Settings)
3. Keep the app **open and active** during download
4. Check for app updates

### "Offline search not finding protocols"

1. The protocol may not be cached ‚Äì try a broader search
2. Offline search uses **keywords**, not AI ‚Äì use specific terms
3. Check that the protocol category is downloaded

### "Outdated protocol warning"

If you see this warning:
1. Connect to the internet when possible
2. The app will automatically update
3. Check the protocol date to ensure accuracy
