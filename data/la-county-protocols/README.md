## File Naming Convention

- `XXXX-ProtocolName.pdf` - Standard protocol document
- `XXXX-P-ProtocolName.pdf` - Paramedic-level supplement/flowchart
- `XXXX.X-ProtocolName.pdf` - Sub-section or form

## Notes

1. **"-P" suffix**: Many protocols have a paramedic version (ending in `-P`) that includes flowcharts and ALS interventions
2. **Protocols are current** as of download date; LA County updates these regularly
3. **RAPID LA County Medic App**: LA County also provides a mobile app version of these protocols
4. **Additional sections not downloaded**: REF 100-700, 900-1100, and 1300 contain administrative policies rather than clinical protocols

## Usage

These protocols are intended for:
- Populating the Protocol Guide database
- Cross-referencing with other county protocols
- Training and education purposes

For the most current versions, always check the official LA County EMS website.
